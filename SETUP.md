# Setup Instructions

## Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- **app/** - Next.js app directory with all pages
- **components/** - Reusable React components (Header, Footer, WhatsAppFloat)
- **public/** - Static assets (images, etc.)

## Pages

- `/` - Homepage with hero section and contact form
- `/about` - About Us page with image carousel
- `/solutions` - Solutions page showcasing industry solutions
- `/products` - Products page displaying RFID products
- `/contact` - Contact page with contact form
- `/software` - Software solutions page
- `/store` - Store page

## Features

✅ Responsive design for mobile, tablet, and desktop
✅ Image carousel on About page
✅ Dropdown navigation menus
✅ WhatsApp floating button
✅ Contact forms
✅ Modern UI with Tailwind CSS
✅ TypeScript for type safety
