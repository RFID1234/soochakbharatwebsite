# Soochak Bharat Website

A modern, responsive website for Soochak Bharat Technologies Private Limited, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Homepage** with hero section and contact form
- **About Us** page with image carousel
- **Solutions** page showcasing various industry solutions
- **Products** page displaying RFID products
- **Contact** page with contact form
- Responsive design for all devices
- WhatsApp floating button for quick contact
- Modern UI with smooth animations

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
